# Publishing Your YoutubeMediaGrabber with Cloudflare Tunnel

This guide explains how to expose your YoutubeMediaGrabber application to the internet using Cloudflare Tunnel, which will provide a random public URL like `https://looking-salad-arm-affects.trycloudflare.com/`.

## Prerequisites

1. Install Cloudflared:
   - **Windows**: Run `winget install --id Cloudflare.cloudflared`
   - **Mac**: Run `brew install cloudflared`
   - **Linux**: Follow the [official installation instructions](https://developers.cloudflare.com/cloudflare-one/connections/connect-apps/install-and-setup/installation/)

## Method 1: Using the Python Script (Recommended)

1. Navigate to the YoutubeMediaGrabber directory where `app.py` is located
2. Run the cloudflare tunnel script:
   ```
   python cloudflare_tunnel.py
   ```
3. The script will:
   - Start the Flask server on port 80
   - Create a Cloudflare tunnel to expose your app
   - Display the random public URL generated by Cloudflare
4. To stop the server and tunnel, press `Ctrl+C`

## Method 2: Manual Setup

If you prefer to set up the tunnel manually:

1. Start the Flask server:
   ```
   python app.py
   ```
   Note: By default, this runs on port 5000. You may need to modify the port in `app.py` to use port 80.

2. In a separate terminal, create a Cloudflare tunnel:
   ```
   cloudflared tunnel --url http://localhost:5000
   ```
   (Use the appropriate port where your Flask app is running)

3. Cloudflare will display a random URL which you can use to access your app from anywhere

## Troubleshooting

- If you get a permission error when using port 80, try using a port number above 1024 (update both the Flask server and tunnel command)
- Make sure there are no firewalls blocking the connection
- Verify that cloudflared is installed correctly by running `cloudflared --version`
